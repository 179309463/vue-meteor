# Create awesome Meteor apps with Vue

Installs the following packages:

 - akryum:vue
 - akryum:vue-component
 - akryum:vue-router
 - akryum:vuex
 - akryum:vue-i18n

You can add more packages:

 - akryum:vue-apollo
 - akryum:vue-i18n-ui
 - akryum:vue-less
 - akryum:vue-sass
 - akryum:vue-jade
 - akryum:vue-coffee
